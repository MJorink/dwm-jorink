# Amber Glow Theme

A warm dark VS Code color theme featuring amber and orange accents.

## Colors

- **Background**: `#000000` - Pure black
- **Accent Dark**: `#e1750f` - Dark orange
- **Accent Light**: `#eaaf25` - Light amber
- **Active Border**: `#fca806` - Bright orange

## Installation

1. Copy this folder to your VS Code extensions directory:
   - **Linux**: `~/.vscode/extensions/`
   - **macOS**: `~/.vscode/extensions/`
   - **Windows**: `%USERPROFILE%\.vscode\extensions\`

2. Restart VS Code

3. Open Command Palette (Ctrl+Shift+P / Cmd+Shift+P)

4. Select "Preferences: Color Theme"

5. Choose "Amber Glow"

## Features

- Deep black background for reduced eye strain
- Warm orange and amber syntax highlighting
- Consistent accent colors throughout the UI
- High contrast for better readability
