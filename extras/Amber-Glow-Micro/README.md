# Amber Glow Micro

A warm colorscheme for the [Micro](https://micro-editor.github.io/) editor with a dark theme.

## Installation

1. Copy `amber-glow-micro.micro` to your micro colorschemes directory:
   - ~/.config/micro/colorschemes/`

2. Set the colorscheme in your `~/.config/micro/settings.json`:
   ```
   {
     "colorscheme": "amber-glow-micro"
   }
   ```
